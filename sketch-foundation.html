About the Project

It is no secret that working in a front-end framework can speed up your workflow, however I set out to make it even faster by emulating Foundation inside Sketch to make future prototypes easier to rapidly design upon. This project was open-sourced on Github and attracted a big following in the designer community.

My Role

My vision was to share this project with as many designers as I could. Since building the project, I've been adding on and maintaining it since it's release.
